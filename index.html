Algorithm Visualizer

Link to the repo

Description

A software made for educationnal purpose. It's aimed to be used by student or curious people learning about algorithm. This program will display the implementation of various algorithm in different popular languages, then run the programm step by step. In order to help people having a better understandinf of the implementation, and the logic behind the algorithm, the software will also plot a graphical representation of those instructions.

vimMotion

Link to the repo

Description

I do not like how Discord forces the user to navigate in the app using scrolling, I prefer using vim keybind. This is exactly what this plugins do, using the Vencord API, I added a new behavior to Discord, making it possible to navigate in a channel using j and k keys. I plan to add more features, like a tooltip to inform the user he's current mode ( normal, or insert mode ) Because after using it, you can forget easily that you are in normal mode and therefore, unable to type.

Stack

  • TypeScript
  • Vencord API
