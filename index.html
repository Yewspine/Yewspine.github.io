What did I do so far ?

Algorithm Visualizer

Link to the repo ( Opens in a new tab )

Description

A software made for educational purpose. It's aimed to be used by student or curious people learning about algorithm. This program will display the implementation of various algorithm in different popular languages, then run the programm step by step.

In order to help people having a better understanding of the implementation, and the logic behind the algorithm, the software will also plot a graphical representation of those instructions.

Stack

  • Java
  • JavaFX
  • Maven

vimMotion

Link to the repo ( Opens in a new tab )

Description

I do not like how Discord forces the user to navigate in the app using scrolling, I prefer using vim keybind. This is exactly what this plugins does using the Vencord API.

I added a new behavior to Discord, making it possible to navigate in a channel using j and k keys. I plan to add more features, like a tooltip to inform the user their current mode ( normal, or insert mode ).

Stack

  • TypeScript
  • Vencord API
